Basic Example of how to use SH_COMMAND_QUIT

Example 1:  Tell all child nodes whose names begin with the letter "J" to go away.

   BMessage msg(SH_COMMAND_QUIT);
   msg.AddString(SH_NAME_TO, "J*");
   root->PostMessage(&msg);

Note:  Another way to accomplish the same thing is to send an SH_COMMAND_REMOVECOMPONENTS message to the parent(s) of the children you wish to delete, and specifying the children to remove in the SH_NAME_CHILDREN field.  Either way is fine.
